## 🧩 Fonctionnalités

- ⛏️ **Mining aléatoire pondéré** (probabilités réelles)
- 💎 **200+ minerais** à drop (et ça peut encore grossir)
- 🌈 **Raretés multiples** (du commun au totalement absurde) :
  - Common, Uncommon, Rare, Epic, Legendary
  - Mythical, Godly, Exotic, Celestial, Heavenly
  - Empyrean, Paradoxal, Primeval, Infinity, Forsaken, Cosmic

- 💾 **Sauvegarde automatique** (LocalStorage)
- 🔍 **Recherche** dans l’inventaire
- ↕️ **Tri** par rareté ou quantité
- 📊 **Statistiques** :
  - Total miné
  - Minéraux uniques
  - Dernier drop

- ⌨️ **Raccourcis clavier** :
  - `M` → Miner
  - `R` → Reset

- 📤 **Export** / 📥 **Import** de sauvegarde (JSON)
- 🔔 **Notifications toast**
- ✨ **UI moderne** (glassmorphism)
- 🌌 **Thème astro animé** (stars + ambiance)

```bash

```

2. Ouvre `index.html` dans ton navigateur.

## 🧠 Philosophie du projet

Ce jeu est un projet fun et expérimental visant à :

- Tester des mécaniques RNG **extrêmes**
- Explorer une UI **futuriste** sans framework
- Construire une base **évolutive** (ajouts de minerais, systèmes, updates…)
